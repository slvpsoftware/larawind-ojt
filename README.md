## Let's Set things up!
1. `npm i`
2. `npm i sass gulp-cli --save-dev`
3. `npm i -G gulp`
4. `npm install -D tailwindcss`
5. `cd app && composer install`

## Start three Servers
1. Gulp - `npm run gulp`
2. Tailwind - `npm run tailwind`
3. Laravel - `cd src && php artisan serve`
